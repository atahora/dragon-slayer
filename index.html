Dragon Slayer

Présentation du code (Dev)

Ce modeste programme a pour but de faire un petit tour du javascript (vanilla) de manière ludique. Cela me permet de présenter mes compétences en programmation orientée objet, intéraction avec le DOM, animations, variables, conditions, boucles etc. Le code source est disponnible sur github !

Règles du jeu

Le jeu est simple. Vous incarnez un chevalier qui doit terrasser un terrifiant dragon.

On choisit d'abord ses paramètres (difficulté, arme et armure) qui influenceront vos chances de gagner.

A chaque tour le gagnant du "pierre-feuille-ciseaux" attaque et cause des dégats à son adversaire.

La partie s'achève lorsque l'un des combattant n'a plus de point de vie

Jouer !
